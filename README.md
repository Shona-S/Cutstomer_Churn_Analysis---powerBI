## Project Overview
Customer churn refers to the loss of customers or subscribers over a specific period. This project focuses on analyzing customer behavior, identifying churn patterns, and providing business insights using Power BI. The dashboard helps organizations understand why customers leave and supports data-driven retention strategies.

## Objectives
- Analyze customer churn trends.
- Identify key factors influencing customer attrition.
- Provide interactive visual insights.
- Support decision-making to improve customer retention.

## Tools & Technologies Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Data Cleaning & Transformation
- Data Visualization Techniques

## Dataset Description
The dataset contains customer-related information such as:
- Customer demographics
- Subscription details
- Service usage information
- Payment methods
- Contract details
- Churn status (Yes/No)

## Dashboard Features

### 1. Churn Overview
- Total Customers
- Total Churn Count
- Churn Percentage
- Active vs Churned Customers

### 2. Demographic Analysis
- Churn based on gender
- Churn based on age group or senior citizen status
- Churn distribution by customer segments

### 3. Service Usage Analysis
- Internet service type analysis
- Add-on services impact (Streaming, Security, Backup, etc.)
- Contract type comparison

### 4. Payment & Billing Analysis
- Monthly charges vs churn
- Payment method analysis
- Contract duration impact on churn

### 5. Interactive Filters
- Region / Location filters
- Contract type filters
- Service type filters

## Key Insights
- Customers with month-to-month contracts show higher churn rates.
- Higher monthly charges tend to increase churn probability.
- Customers using certain service combinations are more likely to stay.
- Payment method and contract duration significantly influence retention.

## DAX Measures Used
- Churn Rate Calculation
- Customer Count
- Retained Customer Count
- Average Monthly Charges
- Service Usage Metrics
